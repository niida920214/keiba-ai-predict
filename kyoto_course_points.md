# 京都競馬場のコース別ポイント

## 芝コース

### 京都 芝1200m
*   **特徴:** 「内前 先行圧倒的に有利」
*   **詳細:** 最初のコーナーまでの距離が短い ＋ 3角が上り坂 → 直線平坦。ペースが上がりにくく、内前圧倒的有利。
*   **激アツ条件:** 1・2枠、前走4角3番手以内（単勝回収率 191%、複勝回収率 114%）

### 京都 芝1400m
*   **特徴:** 「外回り 差し有利の舞台」
*   **詳細:** 芝1200mよりスタートが200m後ろに。3コーナーの上り坂でペースが緩み、長い直線を意識してスローになりやすい。
*   **狙い目:** 追走力に課題のある距離短縮馬。スローからの瞬発力勝負になるため決め手が重要。
*   **データ:** 上がり最速馬 勝率35.6% 単回130% 複回141%

### 京都 芝1600m ＆ 1800m
*   **特徴:** 「距離短縮＆差し馬が狙い目」
*   **詳細:** 3コーナーの上り坂でペースが緩み、長い直線を意識してスローペースに。
*   **狙い目:** 追走力に課題のある距離短縮馬。スローからの瞬発力勝負。
*   **データ（芝1600m外）:** 上がり最速馬 勝率38.4% 単回176% 複回187%
*   **ポイント:** キレが武器の中距離馬や距離短縮馬に注意。

### 京都 芝2000m
*   **特徴:** 「基本は内前も、秋華賞は差し有利」
*   **詳細:** 1コーナーまでの距離が短い ＋ 1、2角が急カーブ ＋ 直線が短い。ペースが上がりにくく基本は内前有利。
*   **特記事項:** 秋華賞だけは差し有利（過去10年の8枠中7頭が3角10番手以降の差し馬。例：昨年のリバティアイランドなど先行勢が早めにスローになった場合）。

### 京都 芝2200m
*   **特徴:** 「タフなコースで距離短縮組有利」
*   **詳細:** これまでの外回り同様に末脚が重要。長い距離 ＋ 3・4角の下りから早めスパートになるためスタミナの要求値が上がる。
*   **狙い目:** 距離短縮馬や、2200m以上の実績がある馬（距離短縮 単勝回収率 133%）。

### 京都 芝2400m
*   **特徴:** 「能力のある馬を信頼すべき」
*   **詳細:** スロー → 瞬発力勝負 ＋ コーナーまで長く枠の有利不利がない。紛れが少なく各馬の能力が発揮されやすい。
*   **データ:** 日経新春杯（ハンデ戦）も1, 2番人気が勝つことが多い（8枠中7頭が1, 2番人気）。

### 京都 芝3000m ＆ 3200m
*   **特徴:** 「中距離馬でも好走可能な長距離戦」
*   **詳細:** スタート後の3角の上り坂でペース緩む ＋ 直線平坦。スタミナの要求値が低く、中距離馬も好走可能。
*   **ポイント:** コーナーを6回通過するため、基本は内枠有利。

## ダートコース

### 京都 ダート1200m
*   **特徴:** 「日本屈指の逃げ・先行有利コース」
*   **詳細:** スタート後の3角の上り坂でペース緩む ＋ 3・4角下り坂 ＋ 直線平坦。
*   **データ:** 逃げ馬 勝率 51.8% 単回115% 複回101%

### 京都 ダート1400m
*   **特徴:** 「芝スタートで外枠有利」
*   **詳細:** 芝部分を長く走れる外枠が有利。3角の上り坂でペースが緩む。
*   **狙い目:** 追走力に課題のある距離短縮馬に注意。

### 京都 ダート1800m
*   **特徴:** 「ダートでは珍しい内枠有利コース」
*   **詳細:** 1角までの距離が短い ＋ 1・2角が急カーブ。外枠がポジションを取れず内枠が有利になる。

### 京都 ダート1900m
*   **特徴:** 「内枠の差し馬を狙え」
*   **詳細:** 初角までの距離が延びてペースUP → 差し有利。1・2角が急カーブで外枠不利 → 内枠有利。
*   **狙い目:** **【内枠の差し馬】**
