# 阪神競馬場のコース別ポイント

## 芝コース

### 阪神 芝2200m（内回り）
*   **特徴:** 「東京でもし負ける様なタフな馬を狙え」
*   **詳細:** 
    *   阪神2000mよりも初角までの距離が長い → ペースが上がりやすい。
    *   内回りコースでスパートのタイミングが早い → 消耗戦になりやすい。
    *   差し馬の台頭が目立ち、東京などの瞬発力勝負で負けた馬がここで一変するケースが多い。
*   **特記事項:** 日本ダービー（東京芝2400m）と宝塚記念（阪神芝2200m）を両方勝利したのは、オルフェーヴルとディープインパクトの2頭のみというタフな条件。

### 阪神 芝2400m（外回り）
*   **特徴:** 「内枠有利・スローになりやすい」
*   **詳細:** スタート地点は阪神2000mと同じだが、こちらは外回り。スローペースからの瞬発力勝負になりやすい。
*   **データ:** 過去5年、1枠・2枠の回収率は共に100%超え。

### 阪神 芝2600m（外回り）
*   **特徴:** 「タフさが問われるコース」
*   **詳細:** スタート地点は阪神2200mと同じ（外回り）。
*   **データ:** 過去5年、逃げ馬の勝利0、上がり最速馬の複勝率94.4%。外回りながら、非常にタフな消耗戦の適性が問われる。

### 阪神 芝3000m（内回り）
*   **特徴:** 「京都3000mとは真逆のスタミナ必須のコース」
*   **（参考）京都3000m:** スタート直後に上り坂＆初角までが短い → ペースが上がる事はほぼ無く、求められるのは「瞬発力・スピード」。
*   **詳細（阪神3000m）:** スタートから下り坂 ＋ 内回りでスパートのタイミングが早い ＋ 直線に急坂がある → 消耗戦になる事がほとんどで、極めて**スタミナの要求値が高いコース**。

## ダートコース

### 阪神ダート全般（大前提）
*   **大前提:** ダートは良馬場の場合には**外枠有利**。
    1.  砂を被らずに済む。
    2.  スムーズにスパートをかける事ができるため。
*   **特徴:** **「阪神ダートは日本一、外枠が有利なコース」**
    *   コーナー角の緩い大箱コースで、外を回されることの負荷（距離ロス）が比較的小さい。
    *   最後の急坂で脚が止まった先行馬が壁になりやすいため、内枠に閉じ込められると圧倒的に不利。
    *   外枠の馬は、3・4コーナーからスムーズにロングスパートをかけやすい分、構造的に恵まれる。

### 阪神 ダート1400m ＆ 2000m
*   **特徴:** 「特に外枠が有利な条件」
*   **詳細:** 
    *   **芝スタート** ＋ 最初のコーナーまでの距離が長い。
    *   芝部分を長く走れる外枠が、スピードに乗って先行争いで有利になる。
    *   ハイペースになりやすいため、先行馬の脚が止まった時に内側にいる馬（内枠）はドン詰まりの不利を受けやすい。
