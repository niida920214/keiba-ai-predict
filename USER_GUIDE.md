# 競馬AI レース予測 使い方ガイド（一般ユーザー向け）

## 1. ログイン

1. 共有されたURLをブラウザで開く
2. 共有されたパスワードを入力してEnter
3. 一度ログインすると、同じブラウザなら**7日間は再入力不要**

## 2. 予測したいレースの「レースID」を調べる

1. [netkeiba](https://race.netkeiba.com) で予測したいレースの**出馬表ページ**を開く
2. ブラウザのアドレス欄を見ると、URLの末尾に `race_id=` という部分がある

   例: `https://race.netkeiba.com/race/shutuba.html?race_id=202605010811`

3. `race_id=` の後ろの**12桁の数字**（この例では `202605010811`）がレースID

## 3. 予測を実行する

1. 「レースID（12桁）」欄にレースIDを入力
2. 「開催日」にそのレースの開催日を選択
3. 「▶ 予測する」を押して**20〜40秒**待つ

> 連続実行はできません（サーバー保護のため15秒のクールダウンがあります）。
> また、出走馬が確定する前（出馬表の発表前）のレースは予測できません。

## 4. 予測結果の見方

### メインの表（各馬の予測）

| 列 | 意味 |
|---|---|
| **予測順位** | AIの評価順（P(Win)が高い順） |
| **馬番** / **馬名** | 出馬表と同じ |
| **P(Win)** | AIが予測した「1着になる確率」。全馬の合計が1になる |
| **P(Place)** | AIが予測した「3着以内に入る確率」（複勝圏内率） |
| **単勝オッズ** | 取得時点の実際の単勝オッズ |
| **人気** | オッズ順の人気順位 |
| **EV** | **期待値**。P(Win) × 単勝オッズ。**1.0を超えていれば「賭け続ければ理論上プラス」**を意味する、このツールで一番重要な指標 |
| **Kelly** | 資金の何割を賭けるのが数学的に最適かの目安（ケリー基準）。大きいほど「厚く賭ける価値がある」 |

### 推奨買い目（表示された場合のみ）

「EV > 1.0 かつ Kelly > 0」＝**AIが市場より高く評価している馬**だけが表示されます。
該当馬がいないレースでは表示されません（＝そのレースには妙味のある単勝がない、という判断）。

### 馬連 / ワイド / 三連複 の上位候補

AI評価上位馬同士の組み合わせについて、的中確率が高い順に表示します。
※ここに出るのは「確率が高い組み合わせ」であって「期待値が高い買い目」ではない点に注意
（人気馬同士の組み合わせはオッズも低いため）。

## 5. 読み方のコツ

- **「勝ちそうな馬」と「買うべき馬」は違う**。P(Win)が一番高い馬でも、オッズが
  低すぎればEVは1.0を下回る（＝買っても長期的には損）。逆に勝率10%でも
  オッズ15倍ならEV=1.5で「買い」になる
- **EVは取得時点のオッズに基づく**。オッズはレース直前まで変動するため、
  購入直前に実際のオッズと見比べること
- 予測に使うオッズは無料APIの都合で**1日数回しか更新されない**ことがあり、
  直前のリアルタイムオッズとズレる場合がある
